Importar archivo del checador

Sube el archivo que exporta el software del BM160 (.xlsx, .xls o .csv). Puede tener cualquier orden de columnas: solo indica cuál es cuál abajo.

Se detectaron 0 filas. Selecciona qué columna corresponde a cada dato. Si fecha y hora vienen juntas en una sola columna, marca la casilla y elige solo esa columna.

Directorio de empleados

Registra los datos de cada empleado. El número de empleado debe coincidir con el ID que usa el checador para poder cruzar las checadas.

Turnos y códigos de pago

Define los horarios de turno y los códigos de pago que puedes asignar a cada empleado.

Configuración general de horario

Se usa como valor por default para retardos y faltas cuando un empleado no tiene un turno específico asignado.

Cualquier hora trabajada en un día por arriba de la jornada esperada se paga como hora extra, multiplicando el sueldo por hora por este factor (ej. 2 = doble).

Resumen por periodo

Elige cualquier rango de fechas y ve un solo empleado o todos a la vez. Suma automáticamente todas las checadas del día (aunque haya 6) en pares entrada/salida.

Calendario de asistencia

Vista mensual de un empleado: verde a tiempo, ámbar retardo, rojo falta.

Reporte por empleado

Elige empleado y rango de fechas para ver horas trabajadas, retardos y faltas día por día.

Los datos (incluidas fotos) se guardan solo en tu navegador (almacenamiento del artefacto), no se comparten con nadie.
